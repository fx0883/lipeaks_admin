## ⚠️ 注意事项

### 数据安全

- **Token 存储**: 使用 localStorage，注意XSS防护
- **敏感信息**: 不要在前端存储敏感数据
- **HTTPS**: 生产环境必须使用 HTTPS

### 性能考虑

- **分页**: 长列表必须使用分页
- **缓存**: 缓存不常变化的数据
- **懒加载**: 图片和组件按需加载
- **防抖**: 搜索等频繁操作使用防抖

### 兼容性

- **浏览器**: 支持现代浏览器
- **移动端**: 考虑响应式设计
- **时区**: 统一转换为本地时间显示
